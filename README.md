# Backend Technical Challenge for Mars Rover 

**Test Input**:  
```
5 5
1 2 N
LMLMLMLMM
3 3 E
MMRMMRMRRM
```

**Expected Output**:  
```
1 3 N
5 1 E
````

To run the program simply do:  

`ruby solution.rb`

and provide the inputs.


## Running test cases for testing

Run `bundle install`  to install gems and then run the command

`bundle exec rspec`
